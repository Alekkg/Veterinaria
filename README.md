# 🐾 PetSystem - Sistema de Gestión Veterinaria

## 📌 Descripción

**PetSystem** es una aplicación web diseñada para la gestión de clínicas veterinarias. Permite administrar mascotas, propietarios y veterinarios mediante una interfaz interactiva y sencilla.

El sistema está desarrollado con tecnologías web básicas y almacenamiento local, ideal como proyecto académico para demostrar habilidades en desarrollo frontend y lógica en JavaScript.

---

## 🚀 Funcionalidades

* 🐶 Gestión de mascotas (crear, editar, eliminar)
* 👤 Gestión de propietarios
* 🩺 Gestión de veterinarios
* 💾 Persistencia de datos usando `localStorage`
* 🔐 Sistema básico de autenticación (login)
* 🔄 Navegación dinámica sin recargar la página



## ⚙️ Instalación y ejecución

1. Clona este repositorio:

```
git clone https://github.com/tu-usuario/petsystem.git
```

2. Abre el proyecto en tu editor (VS Code, NetBeans, etc.)

3. Ejecuta el archivo:

```
index.html
```
